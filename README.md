# skills                                                                                                 
                                                                                                                  
  A collection of custom slash commands for [Claude Code](https://claude.ai/code).                                
                                                                                                                  
  ## Installation                                                                                                 
                                                                                                                  
  Copy any `.md` file from this repo into your project's `.claude/commands/` directory (or `~/.claude/commands/`  
  for global access).
                                                                                                                  
  ## Usage        

  Each file becomes a slash command named after the file:                                                         
   
  /refactor                                                                                                       
  /your-skill-name
                                                                                                                  
  ## Skills
                                                                                                                  
  | Command | Description |
  |---------|-------------|
  | `/refactor` | Refactor a Flutter project to Clean Architecture with Cubit-driven navigation |
